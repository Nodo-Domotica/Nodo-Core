  Dit is de beheertool voor de Nodo. Het is een grafische omgeving die 
  commando's ontvangt en verstuurd via de seriële poort van de computer. 

  Voorwaarden:
  - Linux, Windows of Macintosh omgeving
  - Java omgeving (JRE of JDK) versie 6.

  Pak het bestand uit in een folder naar keuze. Start vanuit die folder met: 
  
      java -jar NodoDueBeheer.jar

  Mocht de communicatie met de Nodo niet lukken, probeer dan een andere 
  architectuur (staan opgesomd in lib) met de optie -arch <arch>.


\**************************************************************************/

  Ontwikkelaars: Installeer Maven, NetBeans en kijk in de support folder.
